# Robo_urdf

This repo built folowing this example https://github.com/joshnewans/urdf_example


## Launch Instructions

Terminal 1:
 ros2 launch <launch_file>
 
 Launch file options:
      rsp_sim.launch.py: run Gazebo Sim with ros2 control
      launch_robot.launch.py: run robot hardware
      
Terminal 2:
  rvz2 -d /path/to/config/robo_view.rviz
